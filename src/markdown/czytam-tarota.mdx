## Overview

This project involved creating a visually-rich and animated frontend for an e-commerce service. The primary goal was to build an engaging and intuitive user experience that guides the user smoothly through the service description, ordering process, and payment. The focus was on modern UI design, fluid animations, and a seamless checkout integration.

## Tech Stack

- • **Language**: Typescript
- • **Frameworks/Libraries**: React, React Router
- • **Styling & UI**: Tailwind CSS, Radix UI, Lucide Icons
- • **Animations**: Framer Motion, GSAP
- • **Payments**: Stripe
- • **Other tools**: Vite, Zod, ESLint, Prettier

## Key Features

- • **Engaging User Experience**: Designed to be intuitive and visually appealing, with a focus on user flow.
- • **Complex Animations**: Implemented scroll-based and interactive animations using Framer Motion and GSAP to enhance the user interface.
- • **Secure Payment Integration**: Integrated Stripe for a seamless and secure checkout process.
- • **Responsive Design**: Fully responsive layout ensuring a great experience on all devices, from desktops to mobile phones.
- • **Interactive Components**: Utilized carousels and sliders for dynamic content presentation.
- • **Form Handling & Validation**: Built a robust order form with client-side validation using Zod.
- • **Modern Tooling**: Developed with a modern frontend stack including Vite for fast builds and ESLint/Prettier for code quality.
