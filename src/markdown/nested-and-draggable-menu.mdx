## Overview

This was the project for the Polish client so the description and some part of the applications are in Polish.

## Tech Stack

- • **Language**: Typescript
- • **Frameworks/Libraries**: React.js, Next.js, Dnd Kit, Zod, React Hook Form
- • **Styling**:
- • **Other tools**: Docker, Husky, Uuid, Eslint, Prettier

## Key Features

- • Add, edit, delete nested menu items
- • Integration tests
- • Re-usable components
- • Form validation and handling
- • Pixel perfect design
- • Drag and drop menu items
