## Overview

This one was an assignment. Although I didn't have more time to spend with it and there is room for improvement, I'm happy with the results.

## Tech Stack

- • **Language**: TypeScript
- • **Frameworks/Libraries**: React, React Router
- • **Styling**: Material UI (MUI), Emotion, MUI X Charts
- • **Other tools**: Vite, ESLint

<br />
## Key Features

- • Charts, tables with mocked data
- • Responsiveness to different screen sizes and devices
- • Documentation of your design and principles in implementation
- • Full functional and styled side menu
- • 404 page
- • Re-usable components
- • Redirect to the /dashboard/sales from home page /
- • Best standards and practices followed for file structure and design implementation
