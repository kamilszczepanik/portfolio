## Overview

I've created an e-commerce website for one of my clients that is based on the project starter created in the course. As of now I can't share the website because of the NDA therefore I share here the base project.

## Solution

I was working close with the client from the idea to the implementation with focus on creating the most efficient e-commerce platform in terms of performance as well as design and sales.

## Tech Stack

- • **Language**: Typescript, Node
- • **Frameworks/Libraries**: React 19, Next.js 15, Prisma, React Hook Form, Next Auth, Jest Testing
- • **Styling**: Tailwind, ShadCN UI
- • **Deployment**: Vercel
- • **Other tools**: Zod, Eslint

## Key Features

- • Next Auth authentication
- • Admin area with stats & chart using Recharts
- • Order, product and user management
- • User area with profile and orders
- • Stripe API integration
- • PayPal integration
- • Cash on delivery option
- • Interactive checkout process
- • Featured products with banners
- • Multiple images using Uploadthing
- • Ratings & reviews system
- • Search form (customer & admin)
- • Sorting, filtering & pagination
- • Dark/Light mode
- • Much more
