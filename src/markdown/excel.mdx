## Overview

Excel sheet clone with 10000x10000 grid.

## Tech Stack

- • **Language**: Typescript
- • **Frameworks/Libraries**: React, Next JS
- • **Styling**: Tailwind
- • **Other tools**: Vercel, Eslint, Prettier

## Key Features

- • 10000 columns, 10000 rows
- • Edit any cell
- • Basic Mathematical operations between columns
- • Performance optimization(rendering only visible cells, re-rendering only modified cells)
- • Formula bar
